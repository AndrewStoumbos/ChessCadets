# ChessKids

A remake of the old ChessKids game built with Unreal Engine.

## Requirements

- Unreal Engine 5.7

## Getting Started

1. Open `ChessKids.uproject` in Unreal Engine 5.7
2. The project will compile and open automatically

## Project Structure

- `Content/` - All game assets and blueprints
- `Config/` - Engine and game configuration files

## Features

- Chess board environment
- Procedural nature spawning system
- Nature pack assets
